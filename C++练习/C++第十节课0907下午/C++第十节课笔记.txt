二. 模板

 1. 函数模板
     template
     typename
     template <typename 类型名>
     正常的函数声明

     函数模板可以传入一个或几个参数的类型,当函数模板被调用的时候会生成指定类型的函数.

     (1). 当普通函数和模板函数匹配程度相同时,优先普通函数.如果模板函数的匹配性更好,那就优先模板函数.
     (2). 函数模板在传参的过程中不存在隐式类型转换.

2. 类模板

   template <class T>
   class 类名
   {
   };
   
   template <class T>
   返回值 类名<T>::函数名(参数列表)
